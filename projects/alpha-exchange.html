What It Does

A distributed benchmarking platform of six Go microservices where users upload order-matching engines, each driven by a bot fleet and ranked on a live leaderboard by throughput, tail latency, and correctness. Runs every untrusted submission in a locked-down container with seccomp, AppArmor, dropped capabilities, and a read-only non-root filesystem. A shadow order book validates every fill against an authoritative reference engine regardless of execution order.

A user uploads a matching engine, a bot fleet drives traffic at it, and the leaderboard ranks it against everyone else. The submissions are untrusted code, so isolation and independent correctness checking are built into the platform.

How It Works

Six Go Microservices

The platform splits into six Go services: submission-api takes uploads, build-worker builds them, orchestrator schedules runs, bot-fleet drives order flow, telemetry-ingester collects metrics, and leaderboard-api ranks results. A React single page app is the frontend. Services talk over gRPC with Protobuf, telemetry flows through Kafka, Redis backs fast lookups, and Docker and Kubernetes run the fleet.

Sandboxed Execution

Every untrusted submission runs in a locked-down container. seccomp restricts the system calls it can make, AppArmor confines its file and resource access, Linux capabilities are dropped, and the filesystem is read-only and non-root. A submission that misbehaves stays boxed in.

Independent Correctness Checking

An authoritative reference matching engine defines the correct result for any order sequence. A shadow order book validates every fill against that reference independent of what the submission reports, so it catches impossible fills regardless of the order in which events execute.

Results

Submissions are scored on a live leaderboard across three axes: throughput, tail latency, and correctness. A fast engine that produces an impossible fill does not climb the board, because the shadow order book flags it independent of the submission-reported result.
