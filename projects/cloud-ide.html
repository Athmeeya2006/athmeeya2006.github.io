What It Does

A multi-user browser IDE with the Monaco editor and a live shell over WebSockets, plus sandboxed execution and a one-click full-stack runner. A multi-tenant backend backs per-project databases across SQLite, PostgreSQL, MySQL, and MongoDB from a few shared servers, with remote-capable live preview.

The project puts a working development environment in the browser. Several users can edit, run a shell, execute code, and preview a running app without installing anything locally, while the backend keeps each project isolated and gives it the database it needs.

How It Works

The Editor and Shell

The frontend is React with TypeScript and embeds the Monaco editor, the same editor core used in VS Code. A live shell streams over WebSockets, so terminal input and output move in real time. Code runs in sandboxed execution, and a one-click full-stack runner brings a project up end to end.

The FastAPI Backend

The backend is FastAPI on Python, split into routers by concern: terminal, preview, git, files, processes, database, projects, and auth. A remote-capable live preview lets the running app be viewed even when it executes away from the browser.

Multi-Tenant Databases

A multi-tenant backend provisions a database per project across SQLite, PostgreSQL, MySQL, and MongoDB, drawing them from a small shared server pool rather than standing up a server per user. One pool of shared servers fronts many isolated per-project databases.

Results

The result is a browser IDE that edits, runs a shell, executes code, and previews a full-stack app for multiple users at once. Per-project databases span four engines from a few shared servers, and live preview works even when the project runs remotely.
