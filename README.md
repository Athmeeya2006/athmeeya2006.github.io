# Athmeeya M Kashyap — Portfolio

My personal portfolio website, live at **[athmeeya2006.github.io](https://athmeeya2006.github.io)**.

A static site (vanilla HTML, CSS, and JavaScript) hosted on GitHub Pages.

## Contents

- **index.html** - the main single-page portfolio.
- **styles.css** - all the site styling.
- **scripts.js** - animations, intro, skills radar, mobile menu, and other interactivity.
- **projects/** - a detail page per project, plus the shared `project-detail.css` and `project-detail.js`.
- **assets/** - downloadable résumés (Quant + SWE) and CV image.
- **.github/workflows/ci.yml** - CI that validates HTML/CSS and checks internal links.
